## 説明
闇を討ってください。<br>
こちらから遊べます。⇒  [WebMSX](http://webmsx.org/?MACHINE=MSX1J&PRESETS=MSXMUSIC&ROM=https://github.com/CoBinee/mayq2-msx/raw/main/rom/MAYQ2.ROM)

## 操作
- `↑`, `↓`, `←`, `→`<br>上下左右に移動
- `ESC`<br>パスワード表示
- ジョイスティックにも対応

## 動作環境
- 16KB 以上の RAM を搭載した MSX

## 開発
- [SDCC](https://sdcc.sourceforge.net)
